for running the website please follow the below steps.
1)Extract the zip folder to your "C:\xampp\htdocs".
2) create a database with name searchus (it's mandatory) in your xampp/phpmyadmin.
3)now select the database searchus and click on import and then select the searchus.sql file.
4)click go.
5)now start Apache and mysql in your xampp controll panel.
6)open your web browser and type "localhost/".
7)The website will open.
